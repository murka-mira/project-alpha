# Dylan's Void

A personal web app built with Next.js — a space for games, tools, and whatever comes next.

## Pages

| Route | Description |
|---|---|
| `/` | Landing page |
| `/dashboard` | Hub with links to all sections and external school tools |
| `/games` | Games library |
| `/games/clicker` | Idle clicker game |
| `/games/explorer` | Void Explorer — retro adventure game (in progress) |
| `/about` | About the Void |

## Features

- Night sky background with twinkling stars across all pages
- **Clicker game** — click to earn points, buy upgrades (auto-cursor, robot, factory, lab), pause/resume, fullscreen mode with shop overlay, explosion particle effects
- **Void Explorer** — retro Zelda-style adventure game, start screen with pixel art character
- Dashboard quick-links to Aries Student Portal and Canvas (IUSD)

## Stack

- [Next.js 16](https://nextjs.org) (App Router, TypeScript)
- [Tailwind CSS v4](https://tailwindcss.com)

## Running locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
